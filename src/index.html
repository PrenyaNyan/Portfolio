Strong Box

This project aims to create the circuitry and program for a safe using an Arduino board, LEDs, and simple electronic components to simulate its operation. The safe's opening process involves multiple steps: activating the electronic circuitry through a combination of switches and, if the combination is correct, prompting a second security code on a keypad to grant access to the safe's contents.

We were tasked with identifying and implementing the most optimized metaheuristic method, then providing proof of its effectiveness.
